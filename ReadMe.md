# `nome-aleatorio`

> Obter um [nome](https://pt.wikipedia.org/wiki/Lista_de_nomes_portugueses) (português) aleátorio.

**Instalação**: `npm install --save nome-aleatorio`.

## Uso

Veja um exemplo:

```javascript
var nomeAleatorio = require('nome-aleatorio');

console.log(nomeAleatorio());
//=> 'Salvador'
```
